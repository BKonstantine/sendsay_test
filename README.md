# Тестовое задание для компании Sendsay

Проект представляет из себя drag-and-drop конструктор, с помощью которого можно собрать калькулятор

## Технологии

* React
* TypeScript
* Redux Toolkit
* React DND
* Framer-motion
* Vite

## Запуск проекта

Установить зависимости
```
npm install
```

Запустить локальный сервер
```
npm run dev
```
